## Install
REQS: Python V3.11.3

First install all the dependencies with:
```
pip install requests discord.py python-dateutil python-dotenv
```

Create a '.env' file inside the directory:
```
TOKEN=(your bot token)
```

Run main.py
```
python3 main.py
```
